How to compile:
1. To run this code your machine should have java installed.
2. run javac WordSuggestion.java command to compile.
3. If you are using linux use "./WordSuggestion" or if you are using windows use "java WordSuggestion" command to
   run the program.
4. Enter the input in the form of argument with the file name
5. Now enter the filepath.
